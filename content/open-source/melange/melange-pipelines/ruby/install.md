Install a ruby gem

### Dependencies
- busybox
- ca-certificates-bundle


### Reference
| Input      | Description                                                     |
|------------|-----------------------------------------------------------------|
| `gem`      | Gem name                                                        |
| `gem-file` | The full filename of the gem to build                           |
| `version`* | Gem version to install. This can be a version tag (1.0.0)       |
| `opts`     | Options to pass to the gem install command Default is set to `` |
